## 文档定位

这不是一组普通 PRD，而是一套 AI 产品经理案例材料。文档重点展示：

- 如何从用户真实购物问题出发定义 AI 产品机会。
- 如何判断哪些环节应该引入 AI，哪些环节必须保留人工确认。
- 如何把多模态识别、图片编辑、embedding、RAG 和 LangGraph 决策流组合成产品闭环。
- 如何识别当前 AI 能力边界，并通过用户确认、状态标记和评估指标降低风险。

## 当前 MVP 实现范围

已实现或已验证：

- Supabase Auth、Postgres、Storage、RLS、pgvector 数据底座。
- 衣橱图片上传、原图保存、Qwen-Image-Edit 展示图生成。
- Qwen-VL 原图 + 展示图双图识别。
- 衣服标签人工确认和 embedding 入库。
- 商品截图识别、衣橱检索、LangGraph 决策报告。
- 决策状态保存和决策清单。
- 聊天页、衣橱页、决策清单页、设置页界面。

规划中：

- 更稳定的图像一致性校验。
- 搭配板图片自动生成。
- 用户反馈学习闭环。
- 更完整的线上部署、内测数据采集和评估面板。
