# 穿搭知识库来源与整合原则 v1

本文件记录第一版本地知识库的资料类型、参考方向和整合原则。知识卡不复制外部原文，只把公开资料、服装识别研究和中文穿搭语境归纳成可检索、可解释、可用于购买决策的知识结构。

## 资料策略

- 覆盖范围：全品类时尚，优先日常高频品类。
- 资料策略：内置知识库 + 联网检索补充。
- 标签颗粒度：中细颗粒，先保证稳定识别和稳定决策。
- 语言：中文为主，保留必要英文概念用于来源追溯。
- 第一版状态：本地 review，不写 Supabase，不生成 embedding。

## 来源类型

### 1. 视觉识别标签来源

- iMaterialist Fashion Attribute Dataset  
  参考地址：https://arxiv.org/abs/1906.05750  
  用途：多标签、细粒度服饰属性体系参考，例如颜色、材质观感、图案、版型等。

- DeepFashion / DeepFashion2  
  参考地址：https://arxiv.org/abs/1901.07973  
  用途：服装检测、分割、姿态、遮挡、品类识别等视觉任务参考。

- Fashion IQ  
  参考地址：https://arxiv.org/abs/1905.12794  
  用途：对话式购物、视觉属性反馈、商品描述和用户自然语言偏好结合的参考。

### 2. 长期衣橱与消费判断

- Capsule wardrobe  
  参考地址：https://en.wikipedia.org/wiki/Capsule_wardrobe  
  用途：少量高复用单品、协调色彩、覆盖多场景的衣橱组织思路。

- Cost-per-wear / wardrobe utilization  
  用途：把价格放到长期穿着次数、维护成本、使用场景中判断，而不是只看折扣和单价。

### 3. 穿搭公式与实用规则

- 三色原则、third piece、3-3-3、five outfit rule  
  参考地址：https://www.washingtonpost.com/home/2026/01/19/third-piece-numerical-fashion-formulas/  
  用途：作为启发式规则，帮助系统解释颜色协调、第三件、旅行 capsule、购买前搭配验证。

这些公式只作为辅助判断，不写成绝对规则；用户风格、场景和舒适度优先。

### 4. 电商与零售信息

- 来源类型：商品品类命名、材质描述、版型表达、护理提示。
- 用途：让视觉识别和前端标签更贴近用户在购物截图中看到的语言。
- 处理方式：只提炼通用命名和风险信号，不复制商品文案。
- 本轮补充重点：Polo 衫、连帽卫衣、打底衫、防晒衫、雪纺衫、羊毛衫/羊绒衫、防晒衣、冲锋衣、软壳衣、抓绒衣、雨壳、工装外套、小香风外套、羊羔毛外套、棉服、休闲裤、小脚裤、牛仔短裤、皮裤、吊带连衣裙、西装连衣裙、旗袍、瑜伽裤、鲨鱼裤、运动裤、户外裤等更贴近真实购物搜索的衣服和裤装品类。
- 同步吸纳为标签的电商高频表达：免烫、加绒、三防、羊毛、羊绒、羊羔毛、鹅绒、小香风、新中式、甜美、度假风、商务、工装风、优雅、辣妹风、高领、V 领、插肩、连帽、无袖、短袖、长袖、中长款、小脚。
- MVP v1 暂不把鞋、包、配饰作为核心识别品类；相关词可在搭配建议中辅助出现，但不进入第一版衣橱入库 taxonomy。

### 5. 中文穿搭语境

- 关键词：通勤、轻熟、韩系、日系、松弛感、老钱风、百搭、显利落、基础款、氛围感。
- 用途：让标签和决策报告符合中文用户的表达习惯。
- 处理方式：把模糊表达转成可控标签，例如把“很百搭”拆成基础款、过渡款、颜色桥梁、场景覆盖。

### 6. 功能服饰与户外分层资料

- REI layering basics  
  参考地址：https://www.rei.com/learn/expert-advice/layering-basics.html  
  用途：补充 base layer、middle layer、shell layer、fleece、down、soft shell、waterproof/breathable 等功能服饰语言。

- REI rainwear guide  
  参考地址：https://www.rei.com/learn/expert-advice/rainwear.html  
  用途：补充防水、防泼水、透气、雨壳、硬壳、城市雨天与户外雨天的区别。

- REI UPF clothing guide  
  参考地址：https://www.rei.com/learn/expert-advice/sun-protection.html  
  用途：补充 UPF、防晒衣、防晒衫、帽子、墨镜、覆盖面积和透气舒适度等判断维度。

- 防晒衣与冲锋衣中文品类资料  
  参考地址：https://zh.wikipedia.org/wiki/%E9%98%B2%E6%99%92%E8%A1%A3  
  参考地址：https://zh.wikipedia.org/wiki/%E5%86%B2%E9%94%8B%E8%A1%A3  
  用途：补充中文语境下“防晒衣”“冲锋衣”从功能服饰进入日常穿搭的品类表达。

## 整合原则

1. 只做归纳改写，不复制外部资料原文。
2. 不使用身材羞辱、绝对审美否定或强制性消费建议。
3. 所有规则都服务于“降低冲动消费、提高衣橱利用率、减少重复购买、形成长期风格”。
4. 知识卡应能被 RAG 检索，也能被用户理解。
5. 视觉识别标签和购买决策标签保持同一套 taxonomy，避免前后端语言不一致。
6. 当知识卡涉及风格或审美时，用“适配、风险、场景、舒适度”表达，不用“好/差”的绝对判断。
